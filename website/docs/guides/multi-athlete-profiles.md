# Multi-Athlete Profiles

Coaches or multi-sport athletes can track **multiple profiles** on a single device. Each profile maintains its own session history, baseline, and verdicts.

## Creating a Profile

1. Go to **Settings** → **Profiles**
2. Tap **Add Profile**
3. Enter the athlete's name (1–100 characters)
4. The profile is created and set as active

## Switching Profiles

1. Go to **Settings** → **Profiles**
2. Tap on the profile you want to switch to
3. The app immediately loads that profile's data, baseline, and verdicts

When you switch profiles:
- The home screen shows the active profile's latest reading and verdict
- History shows only the active profile's sessions
- Baseline is computed from the active profile's data only

## Sharing Verdicts

Each profile can share its latest verdict via the device's native share sheet:

```
🟢 Go Hard — rMSSD: 52.3 ms (106% of baseline)
Streak: 14 days | April 20, 2026
```

This generates a formatted text summary suitable for messaging, social media, or team chats.

## Profile Management

- **Rename**: Tap the edit icon next to a profile name
- **Delete**: Swipe left (iOS) or long-press (Android) on a profile. **Sessions are retained** in the database — only the profile metadata is removed
- **Active indicator**: The currently active profile is highlighted with a checkmark

## Use Cases

### Coach with Multiple Athletes
Create a profile for each athlete you work with. Before their morning reading, switch to their profile so the data goes to the right person.

### Multi-Sport Athlete
If you train in very different modalities (e.g., cycling + BJJ), you might want separate profiles to track how each sport affects your recovery independently.

### Family Sharing
Multiple family members using the same device can each have their own profile with independent baselines and history.
